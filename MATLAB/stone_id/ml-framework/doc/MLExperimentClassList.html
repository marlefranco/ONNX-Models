ML-Experiment Class List

Configure and run the following experiment types below. Please see Getting Started for an overivew of the syntax and workflow.

Types

  • Supervised
  • experiment.Classification
  • experiment.Regression
  • SemiSupervised*
  • experiment.SemiSupervised (Classification)
  • Unsupervised*
  • experiment.Cluster
  • Predictive Maintenance
  • experiment.RUL (Degradation)
*Classes are in-progress and/or undocumented.

Syntax

Create an experiment. See Getting Started for more detail / next steps.
session = experiment.Regression(...
"Data", thisData, ...
"DataFcn", @(x, settings)thisPipeline(x, settings{:}), ...
"Model", thisModel, ...
"DataConfiguration", theseParameters, ...
"ModelConfiguration", theseParameters, ...
"Description", "Some description of experiment" )

Getting help

For documentation on:
% the package
help experiment
% a selected class
help experiment.Regression
% a method of the class
help experiment.Classification/somemethod
% to generate a list of methods
methods experiment.Regression
